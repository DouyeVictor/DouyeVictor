Bike Share Project

This project was part of the individual capstone project as provided for the Google Data Analytics Professional Certificate. According to the project brief, the aim of this project is to identify the difference that exists between casual riders and subscribers(members). This task underlies the hypothesis that members contribute more for the Bike Share company. The dataset has been cleaned and analyzed using R. Visualizations have been made with Tableau
